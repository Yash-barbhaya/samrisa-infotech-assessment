# Samrisa Infotech Technical Assessment

This repository contains the complete full-stack solution for the technical assessment tasks.

## 📁 Project Structure
* **`/Frontend`** - Contains the checkout user interface built using vanilla HTML, CSS, and JavaScript. Enforces interactive form-state validations (dirty tracking).
* **`/Backend`** - Contains the ASP.NET Core Web API template built with .NET 8, establishing endpoint configuration, CORS policies, data models (DTOs), and server-side verification.
* **`SortingSolution.md`** - Contains the algorithmic pseudocode breakdown for executing a Bubble Sort on a Singly-Linked List by swapping physical nodes.

## 🚀 Execution Instructions

### 1. Run the Backend API
1. Navigate to the backend application folder: `cd Backend/SamrisaBackend`
2. Start the API application host profile: `dotnet run --launch-profile https`
3. The server will launch locally at `https://localhost:7157`

### 2. Run the Frontend UI
* Navigate to the `Frontend/` directory and open `index.html` inside any standard modern browser web environment (or run via the VS Code Live Server utility extension).
